### 간단 요약

- **필수**: 모든 커밋 전 README.md 업데이트
- **형식**: `[타입] 설명` (예: `[solve] 프로그래머스 lv1 문제명`)
- **타입**: solve, review, refactor, docs, chore, fix, add
- **자동화**:
  - Pre-commit Hook: README.md 업데이트 확인
  - Cursor Rules (`.cursorrules`): AI가 자동으로 README.md 업데이트 및 커밋 도움
