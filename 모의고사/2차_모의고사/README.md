## 💡 각 문제별 힌트 (막힐 때만 보기!)

### 문제 1: 예상 대진표

<details>
<summary>힌트 보기</summary>

- 매 라운드마다 번호가 절반으로 줄어듦
- `Math.ceil(n / 2)` 활용
- 두 번호가 같아질 때까지 반복
</details>

### 문제 2: 영어 끝말잇기

<details>
<summary>힌트 보기</summary>

- 중복 체크: `Set` 사용
- 끝말잇기 체크: `word1[끝] === word2[처음]`
- 사람 번호: `(i % n) + 1`
- 차례: `Math.floor(i / n) + 1`
</details>

### 문제 3: 카펫

<details>
<summary>힌트 보기</summary>

- 전체 크기 = brown + yellow
- 약수를 구해서 가로 × 세로 조합 찾기
- 조건: `(가로-2) × (세로-2) === yellow`
- 가로 >= 세로
</details>

### 문제 4: 튜플

<details>
<summary>힌트 보기</summary>

- 문자열 파싱: `replace`, `split` 활용
- 길이 순으로 정렬
- 이전 집합과의 차이를 찾기 (Set 활용)
</details>
