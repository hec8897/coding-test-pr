# 예상 대진표

## 📌 문제 정보
- **레벨**: Lv.2
- **목표 시간**: 15분
- **개념**: 구현, 시뮬레이션, 수학

---

## 📝 문제 설명

△△ 게임대회가 개최되었습니다. 이 대회는 N명이 참가하고, 토너먼트 형식으로 진행됩니다.

- N명의 참가자는 1번부터 N번까지 번호가 부여됩니다.
- 1라운드에서는 1번↔2번, 3번↔4번, ... 이런 식으로 붙게 됩니다.
- 각 라운드에서 이긴 사람은 다음 라운드에 진출합니다.
- 다음 라운드에서도 번호 순서대로 경기가 진행됩니다.

예를 들어 8명이 참가하는 경우:

```
1라운드: (1,2), (3,4), (5,6), (7,8)
         이긴 사람: 2, 3, 5, 7

2라운드: (2,3), (5,7)
         → 번호를 재배정하면: (1,2), (3,4)
         이긴 사람: 3, 5

3라운드: (3,5)
         → 번호를 재배정하면: (1,2)
         이긴 사람: 3 (우승)
```

게임 참가자 수 N, 참가자 번호 A, 경쟁자 번호 B가 함수 solution의 매개변수로 주어질 때,  
**A번 참가자와 B번 참가자가 몇 라운드에서 만나는지** 구하세요.

---

## 🎯 제한사항

- N : 2^1 이상 2^20 이하인 자연수 (2의 지수 승)
- A, B : N 이하인 자연수 (A ≠ B)
- 항상 A와 B는 서로 만날 때까지 이긴다고 가정합니다.

---

## 📊 입출력 예

| N | A | B | answer |
|---|---|---|--------|
| 8 | 4 | 7 | 3 |

### 입출력 예 설명

**예제 #1**

```
1라운드:
- (1,2), (3,4), (5,6), (7,8)
- 4번과 7번은 각자 이김
- 다음 라운드 번호: 4 → 2, 7 → 4

2라운드:
- (1,2), (3,4)
- 4번과 7번은 각자 이김
- 다음 라운드 번호: 2 → 1, 4 → 2

3라운드:
- (1,2)
- 4번과 7번이 드디어 만남!

답: 3
```

---

## 💡 핵심 개념

### 패턴 찾기
```
번호 변화 규칙:
- 홀수: (n + 1) / 2
- 짝수: n / 2

또는 Math.ceil(n / 2)로 통일 가능!
```

### 예시
```
1라운드: A=4, B=7
→ A = Math.ceil(4/2) = 2
→ B = Math.ceil(7/2) = 4

2라운드: A=2, B=4
→ A = Math.ceil(2/2) = 1
→ B = Math.ceil(4/2) = 2

3라운드: A=1, B=2
→ A = Math.ceil(1/2) = 1
→ B = Math.ceil(2/2) = 1
→ A === B! (만남)
```

---

## 🔑 풀이 전략

1. A와 B가 같아질 때까지 반복
2. 매 라운드마다:
   - A = Math.ceil(A / 2)
   - B = Math.ceil(B / 2)
   - 라운드 카운트 증가
3. A === B가 되면 현재 라운드 반환

---

## ⏱️ 시간 복잡도

- **O(log N)**: 매 라운드마다 참가자가 절반으로 줄어듦

```typescript
console.log(solution(8, 4, 7));  // 3
console.log(solution(8, 2, 3));  // 1 (1라운드에 바로 만남)
console.log(solution(16, 1, 16)); // 4
console.log(solution(8, 1, 8));  // 3
```
