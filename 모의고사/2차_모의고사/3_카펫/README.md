# 카펫

## 📌 문제 정보
- **레벨**: Lv.2
- **목표 시간**: 40분
- **개념**: 완전탐색, 수학, 약수

---

## 📝 문제 설명

Leo는 카펫을 사러 갔다가 아래 그림과 같이 중앙에는 노란색으로 칠해져 있고 테두리 1줄은 갈색으로 칠해져 있는 격자 모양 카펫을 봤습니다.

```
갈색(brown) 테두리
┌─────────────┐
│ ■ ■ ■ ■ ■ ■ │
│ ■ □ □ □ □ ■ │
│ ■ □ □ □ □ ■ │  □ = 노란색(yellow)
│ ■ ■ ■ ■ ■ ■ │  ■ = 갈색(brown)
└─────────────┘
```

Leo는 집으로 돌아와서 아래 그림처럼 카펫의 노란색과 갈색으로 칠해진 격자의 개수를 기억했지만, 전체 카펫의 크기는 기억하지 못했습니다.

Leo가 본 카펫에서 갈색 격자의 수 brown, 노란색 격자의 수 yellow가 매개변수로 주어질 때  
**카펫의 가로, 세로 크기를 순서대로 배열에 담아** return 하세요.

---

## 🎯 제한사항

- 갈색 격자의 수 brown은 8 이상 5,000 이하인 자연수입니다.
- 노란색 격자의 수 yellow는 1 이상 2,000,000 이하인 자연수입니다.
- 카펫의 가로 길이는 세로 길이와 같거나, 세로 길이보다 깁니다.

---

## 📊 입출력 예

| brown | yellow | return |
|-------|--------|--------|
| 10 | 2 | [4, 3] |
| 8 | 1 | [3, 3] |
| 24 | 24 | [8, 6] |

## 💡 핵심 개념

### 수학 관계식

```
전체 크기 = brown + yellow = 가로 × 세로
노란색 크기 = (가로 - 2) × (세로 - 2)
```

### 완전탐색 (약수)

```typescript
// 전체 크기의 약수를 구함
const total = brown + yellow;

for (let height = 3; height <= total; height++) {
  if (total % height === 0) {
    const width = total / height;
    
    // 조건 체크
    if (width >= height) {  // 가로 >= 세로
      // 노란색이 맞는지 확인
      if ((width - 2) * (height - 2) === yellow) {
        return [width, height];
      }
    }
  }
}
```

---

## 🔑 풀이 전략

### 방법 1: 완전탐색 (약수)
1. 전체 크기 = brown + yellow
2. 세로(height)를 3부터 전체 크기까지 반복
3. 전체 크기가 height로 나누어떨어지면:
   - 가로(width) = 전체 / 세로
   - 가로 >= 세로 조건 확인
   - (가로-2) × (세로-2) === yellow 확인
4. 조건을 만족하면 [가로, 세로] 반환

### 방법 2: 효율적인 탐색
```typescript
// 세로를 √total까지만 탐색
for (let height = 3; height * height <= total; height++) {
  if (total % height === 0) {
    const width = total / height;
    // 체크...
  }
}
```

## 🎯 예시 풀이 과정

### brown = 10, yellow = 2

```
1. total = 10 + 2 = 12

2. 약수 찾기:
   height = 3: 12 % 3 = 0 → width = 12 / 3 = 4
   → (4-2) × (3-2) = 2 × 1 = 2 ✅
   → 답: [4, 3]
```

### brown = 24, yellow = 24

```
1. total = 24 + 24 = 48

2. 약수 찾기:
   height = 3: width = 16 → (16-2) × (3-2) = 14 ❌
   height = 4: width = 12 → (12-2) × (4-2) = 20 ❌
   height = 6: width = 8 → (8-2) × (6-2) = 24 ✅
   → 답: [8, 6]
```

## ⏱️ 시간 복잡도

- **O(√n)**: n은 전체 크기 (brown + yellow)
- 약수를 찾는 과정
