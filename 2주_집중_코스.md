### 📚 학습 내용 (30분)

**DFS (깊이 우선 탐색) 핵심**

```typescript
// DFS 기본 패턴
function dfs(node, visited) {
  // 1. 종료 조건
  if (종료조건) return;

  // 2. 방문 처리
  visited[node] = true;

  // 3. 인접 노드 탐색
  for (let next of graph[node]) {
    if (!visited[next]) {
      dfs(next, visited);
    }
  }
}
```

**개념**:

- 깊이 우선 = 끝까지 가본 후 돌아오기
- 재귀로 구현
- visited로 중복 방지

### 📚 학습 내용 (30분)

**BFS (너비 우선 탐색) 핵심**

```typescript
// BFS 기본 패턴
function bfs(start) {
  const queue = [start];
  const visited = new Set([start]);

  while (queue.length > 0) {
    const node = queue.shift(); // FIFO!

    // 인접 노드 탐색
    for (let next of graph[node]) {
      if (!visited.has(next)) {
        visited.add(next);
        queue.push(next);
      }
    }
  }
}
```

**DFS vs BFS**:

- DFS: 깊이 우선, 재귀, 모든 경로
- BFS: 너비 우선, 큐, 최단거리

```typescript
// 상하좌우 이동
const dx = [-1, 1, 0, 0];
const dy = [0, 0, -1, 1];

function dfs(x, y, grid, visited) {
  // 범위 체크
  if (x < 0 || x >= n || y < 0 || y >= m) return;
  if (visited[x][y]) return;
  if (grid[x][y] === 0) return; // 벽

  visited[x][y] = true;

  // 상하좌우 탐색
  for (let i = 0; i < 4; i++) {
    dfs(x + dx[i], y + dy[i], grid, visited);
  }
}
```

### 📚 학습 내용 (40분)

**DP (동적 계획법) 핵심**

```typescript
// DP의 핵심: 이전 결과를 저장해서 재사용

// 예시: 피보나치
function fib(n) {
  const dp = [0, 1];

  for (let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2]; // 점화식!
  }

  return dp[n];
}
```

**DP vs 재귀**:

- 재귀: 중복 계산 많음 (느림)
- DP: 한 번만 계산 (빠름)

**DP 풀이 3단계**:

1. 점화식 세우기 (제일 중요!)
2. 초기값 설정
3. 반복문으로 채우기

```typescript
// 2차원 DP 패턴
const dp = Array(n)
  .fill(0)
  .map(() => Array(m).fill(0));

// 초기값
dp[0][0] = 시작값;

// 채우기
for (let i = 0; i < n; i++) {
  for (let j = 0; j < m; j++) {
    dp[i][j] = Math.max(
      dp[i - 1][j], // 위에서
      dp[i][j - 1] // 왼쪽에서
    );
  }
}
```

### 목표: 이진 탐색 마스터

```
문제: 1시간 30분
```

### 📚 학습 내용 (30분)

**이진 탐색 핵심**

```typescript
function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    // 등호 중요!
    const mid = Math.floor((left + right) / 2);

    if (arr[mid] === target) return mid;
    if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }

  return -1;
}
```

**핵심**:

- 정렬된 배열 필수
- O(log n) 시간 복잡도
- left <= right (등호!)

```typescript
function backtrack(chosen, candidates) {
  // 종료
  if (chosen.length === n) {
    result.push([...chosen]);
    return;
  }

  // 선택
  for (let i = 0; i < candidates.length; i++) {
    chosen.push(candidates[i]); // 선택
    backtrack(chosen, candidates); // 탐색
    chosen.pop(); // 복구!
  }
}
```

```typescript
  if (n === 1) {
    console.log(`${from} → ${to}`);
    return;
  }

  hanoi(n - 1, from, via, to); // 1. n-1개를 보조로
  console.log(`${from} → ${to}`); // 2. 가장 큰 것을 목표로
```
