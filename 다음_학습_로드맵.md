**이유**: 빵/스티커 문제가 DP였을 가능성, 면접에서 자주 나옴

**학습 목표**:

- 메모이제이션 vs 타뷸레이션
- 점화식 세우기
- 대표 문제 3개 풀기

**기본 개념**:
```typescript
// DP의 핵심: "이전 결과를 저장해서 재사용"

// 예시: 피보나치
const dp = [0, 1];
for (let i = 2; i <= n; i++) {
  dp[i] = dp[i-1] + dp[i-2];  // 이전 값 재사용!
}
```

## 🎯 Priority 2: 중요 (2주일)

### 3️⃣ 이진 탐색 (Binary Search) ⭐⭐

**이유**: 효율성 문제에서 자주 필요, 면접 단골 질문

**학습 목표**:
- 이진 탐색 원리 이해
- while 조건 (left <= right)
- mid 계산 (오버플로우 주의)
- 응용 (parametric search)

**기본 패턴**:
```typescript
function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;
  
  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    
    if (arr[mid] === target) return mid;
    if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }
  
  return -1;
}
```

**추천 문제**:

### 4️⃣ 백트래킹 심화 ⭐⭐

**이유**: 조합/순열 문제, 완전탐색 고급

**학습 목표**:
- 백트래킹 원리 (선택 → 탐색 → 복구)
- 가지치기 (pruning)
- 조합 vs 순열 차이
- N-Queen 패턴

**기본 패턴**:
```typescript
function backtrack(state, choices) {
  // 종료 조건
  if (조건) {
    결과저장();
    return;
  }
  
  // 선택지들 탐색
  for (let choice of choices) {
    // 선택
    state.push(choice);
    
    // 탐색
    backtrack(state, newChoices);
    
    // 복구 (핵심!)
    state.pop();
  }
}
```

**추천 문제**:
```

- 백준: N과 M 시리즈 (15649~)
```
