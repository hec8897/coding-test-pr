# [Lv.2] 괄호 회전하기

## 📌 문제 설명

다음 규칙을 지키는 문자열을 올바른 괄호 문자열이라고 정의합니다.

```
(), [], {} 는 모두 올바른 괄호 문자열입니다.
만약 A가 올바른 괄호 문자열이라면, (A), [A], {A} 도 올바른 괄호 문자열입니다.
예를 들어, [] 가 올바른 괄호 문자열이므로, ([]) 도 올바른 괄호 문자열입니다.
만약 A, B가 올바른 괄호 문자열이라면, AB 도 올바른 괄호 문자열입니다.
예를 들어, {} 와 ([]) 가 올바른 괄호 문자열이므로, {}([]) 도 올바른 괄호 문자열입니다.
```

대괄호, 중괄호, 그리고 소괄호로 이루어진 문자열 `s`가 매개변수로 주어집니다. 이 `s`를 **왼쪽으로 x (0 ≤ x < (s의 길이)) 칸만큼 회전**시켰을 때 `s`가 올바른 괄호 문자열이 되게 하는 x의 개수를 return 하도록 solution 함수를 완성해주세요.

---

## 제한사항

- s의 길이는 1 이상 1,000 이하입니다.

---

## 입출력 예

| s          | result |
| ---------- | ------ |
| `"[](){}"` | 3      |
| `"}]()[{"` | 2      |
| `"[)(]"`   | 0      |
| `"}}}"`    | 0      |

---

## 입출력 예 설명

**예제 #1**

```
"[](){}" 를 회전시키면:

0칸: "[](){}"  ✅ 올바름
1칸: "](){}["  ❌
2칸: "(){}[]"  ✅ 올바름
3칸: "){}[]("  ❌
4칸: "{}[]()"  ✅ 올바름
5칸: "}[](){"  ❌

→ 3개
```

**예제 #2**

```
"}]()[{" 를 회전시키면:

0칸: "}]()[{"  ❌
1칸: "]()[{}"  ❌
2칸: "()[{}]"  ✅ 올바름
3칸: ")[{}]("  ❌
4칸: "[{}]()"  ✅ 올바름
5칸: "{}]()[{" ❌

→ 2개
```

**예제 #3**

```
"[)(]" 는 어떻게 회전시켜도 올바른 괄호 문자열이 될 수 없습니다.
```

---

## 💡 힌트

### 1. 회전 방법
```typescript
// 왼쪽으로 1칸 회전
"[](){}" → "](){}["

// 구현 방법
s = s.slice(1) + s[0]
// 또는
s = s.substring(1) + s.charAt(0)
```

### 2. 올바른 괄호 확인 (Stack 사용)
```typescript
// Stack으로 괄호 매칭 확인
// 여는 괄호: push
// 닫는 괄호: top과 매칭되면 pop
```

### 3. 전체 흐름
```
1. 0부터 s.length-1까지 회전
2. 각 회전마다 올바른 괄호인지 확인
3. 올바르면 count++
```

---

## 🎯 접근 방법

1. **회전 함수**: 문자열을 왼쪽으로 x칸 회전
2. **검증 함수**: Stack을 사용하여 올바른 괄호인지 확인
3. **반복**: 모든 회전(0 ~ s.length-1)에 대해 검증
