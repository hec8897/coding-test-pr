# [Lv.2] 예상 대진표

## 📌 문제 설명

△△ 게임대회가 개최되었습니다. 이 대회는 N명이 참가하고, 토너먼트 형식으로 진행됩니다. N명의 참가자는 각각 1부터 N번을 차례대로 배정받습니다. 그리고, 1번↔2번, 3번↔4번, 5번↔6번, ... 이런 식으로 1라운드에서 대결합니다.

각 라운드에서 이긴 사람은 다음 라운드에 진출하며, 다음 라운드에 진출한 참가자의 번호는 다시 1번부터 N/2번을 차례대로 배정받습니다.

만약 1번↔2번 끼리 겨루는 게임에서 2번이 승리했다면 다음 라운드에서 1번을 부여받고, 3번↔4번에서 겨루는 게임에서 3번이 승리했다면 다음 라운드에서 2번을 부여받게 됩니다.

게임 참가자 수 `n`, 참가자 번호 `a`, 경쟁자 번호 `b`가 함수 solution의 매개변수로 주어질 때, 참가자 `a`와 경쟁자 `b`는 **몇 라운드에서 만나는지** return 하는 solution 함수를 완성해 주세요.

**단, a번 참가자와 b번 참가자는 서로 붙게 되기 전까지 항상 이긴다고 가정합니다.**

---

## 제한사항

- N : 2^1 이상 2^20 이하인 자연수 (2의 지수 승으로 주어지므로 부전승은 없음)
- a, b : N 이하인 자연수 (a ≠ b)

---

## 입출력 예

| N | A | B | answer |
|---|---|---|--------|
| 8 | 4 | 7 | 3      |

---

## 입출력 예 설명

**8명이 참가하는 토너먼트 (A=4, B=7)**

### 1라운드
```
1번 vs 2번  →  승자: 1번
3번 vs 4번  →  승자: 2번  ← A(4번)
5번 vs 6번  →  승자: 3번
7번 vs 8번  →  승자: 4번  ← B(7번)
```

### 2라운드
```
1번 vs 2번  →  승자: 1번  ← A
3번 vs 4번  →  승자: 2번  ← B
```

### 3라운드
```
1번 vs 2번  ← A와 B가 만남!
```

**답: 3라운드**

---

## 💡 힌트

### 핵심 규칙

```
다음 라운드 번호 = 현재 번호를 2로 나누고 올림

예시:
1번 → Math.ceil(1/2) = 1번
2번 → Math.ceil(2/2) = 1번
3번 → Math.ceil(3/2) = 2번
4번 → Math.ceil(4/2) = 2번
5번 → Math.ceil(5/2) = 3번
6번 → Math.ceil(6/2) = 3번
7번 → Math.ceil(7/2) = 4번
8번 → Math.ceil(8/2) = 4번
```

### 알고리즘

```
1. a와 b가 같아질 때까지 반복
2. 각 라운드마다:
   - a = Math.ceil(a / 2)
   - b = Math.ceil(b / 2)
   - round++
3. a === b가 되면 그 라운드에서 만남!
```

---

## 🎯 접근 방법

**시뮬레이션**
- 매 라운드마다 번호를 2로 나누고 올림
- 두 번호가 같아지면 그 라운드에서 만남

**시간 복잡도: O(log N)**
