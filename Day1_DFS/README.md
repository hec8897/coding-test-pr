## 🚀 빠른 시작

```bash

node Day1_연습1_탐색순서.js
```

### 2. 문제 2, 3 풀기

- TypeScript 파일을 열어서 TODO 부분 작성
- 실행: `node Day1_연습2_섬찾기.js`

### 3. 정답 확인

```bash
node Day1_정답.js
```

- **DFS (깊이 우선 탐색)**: 끝까지 가본 후 돌아오는 탐색
- **visited 배열**: 무한 루프 방지, 중복 방문 방지
- **재귀 호출**: 자기 자신을 호출하는 패턴
- **그래프 표현**: 인접 리스트 사용
